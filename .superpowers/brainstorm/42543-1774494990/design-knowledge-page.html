Key UI Features

Suggested Questions

Pre-populated from commonly asked topics. Clickable chips that fill the input.

Source Citations

Every answer shows which document + section it came from. Clickable link to the document in /documents.

Confidence Indicator

Purple border = from your docs. Amber border + ⚠️ = general knowledge, not from your documents.

Streaming Responses

SSE streaming like the existing assistant. Typing indicator while generating.

Session History

Q&A pairs persist in-session (like existing assistant). Clear button to reset.

Index Status Badge

Shows "42 documents indexed" + green Ready badge. Amber if indexing in progress.

Navigation

Add to nav-config.ts under the Operations section, between existing items. Icon: BookOpen from lucide-react. Available to all roles (all staff should be able to look up policies).
